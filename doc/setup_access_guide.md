# Setup & Access Guide

## Overview
This document provides instructions for reviewers to access and test the deployed AI Fitness Companion application. The system consists of a mobile frontend (Expo/React Native) and a cloud-hosted backend deployed via Docker.

At the current stage, the application is distributed as an Android APK and connects to a publicly accessible backend endpoint.

## System Requirements

- Android device **OR** Android emulator (recommended: Android Studio Emulator)
- Stable internet connection
